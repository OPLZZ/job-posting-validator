== job-posting-validator

This is a validator and preview for {RDFa-encoded}[http://www.w3.org/TR/rdfa-in-html/] data described using {Schema.org}[http://schema.org]'s {+JobPosting+}[http://schema.org/JobPosting].

=== Usage

* Run +rake validator:fuseki:init+ to start the database ({Fuseki server}[http://jena.apache.org/documentation/serving_data/index.html]) and load background data.
* Deploy the Rails app.
