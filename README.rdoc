== job-posting-validator

This is a validator and preview for {RDFa-encoded}[http://www.w3.org/TR/rdfa-in-html/] data described using {Schema.org}[http://schema.org] {JobPosting}[http://schema.org/JobPosting].

=== Usage

* Run <tt>rake validator:fuseki:init</tt> to start the database ({Fuseki server}[http://jena.apache.org/documentation/serving_data/index.html]) and load background data.
* Deploy the Rails app.
* Other useful Rake tasks (run <tt>rake -T validator</tt>):
  * <tt>rake validator:fuseki:stop</tt> to stop the Fuseki server.
  * <tt>rake validator:data:import_background_data</tt> to download background data, including the Schema.org vocabulary.
